### Snapshots service

All snapshots for Quark-1 tesnet are available at: https://snapshot.quark.ntrn.info
Use `wget -O - https://snapshot.quark.ntrn.info/<snapshot name> | lz4 -d | tar -xvf -` command in order to download and unpack snapshot.
